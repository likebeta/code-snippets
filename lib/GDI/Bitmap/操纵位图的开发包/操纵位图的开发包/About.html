Backdrop


    可能有的朋友到现在还没弄明白这个软件到底是个什么东西，不要紧，下面我就
详细的解释一下。
    我提供给您的是一个用C++语言编写的一个类，包括类的定义文件（BmpProc.h）
和类成员函数的源代码文件（BmpProc.h）。开发平台是Windows 95 + VC++5.0。所
以如果您不编写Windows程序或者使用的编译器不是VC++（必须是VC++4.2或更高）那
么这个类对您可能没有什么用处。:-(
    以下是这个类的功能：该类主要是完成一些有关位图的操作，包括将位图文件读
入内存、将位图显示在窗口中、以及保存位图到指定的文件当中等等。举个例子来说
，假如你想用磁盘中一个位图文件作为窗口的背景，那么你就得先将这个文件读入内
存，然后把它选入设备描述表（DC），再在背景刷新时将它显示出来。说起来好像很
容易，但实际上是很麻烦得，如果你使用这个类的话，完成以上这些操作只需调用一
两个成员函数即可，非常方便。而且使用它也很容易，只需将这两个文件添加到你的
程序项目当中就行了（具体的使用方法请参阅总目录中“使用”一章）
    说了这么多，你可能对这个类有了一个大概的了解，现在我就解释一下编写这个
类的原因，坦白的说，我之所以编写这个类是因为我对MFC中CBitmap这个位图类很不
满意，或者说是有些失望。它并没有提供多少有用的功能，如果使用 API 函数的话
不见得比使用这个类麻烦多少。于是经过一个多星期的奋斗终于搞出来了位图类
CBmpProc ver1.03，之后又经过多次改进、完善，最终形成了你看到的CBmpProc ver
2.4。我自己用着感觉不错，很顺手。但同时我也知道这里面肯定有欠缺甚至是错误
的地方，所以希望哪位“同志”如果发现了错误或有更好的主意请一定和我联系（联
系方法请见本章目录“联系方法”一节）
