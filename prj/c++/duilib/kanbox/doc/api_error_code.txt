//auth
invalid_request	请求参数不全，或含有错误参数
invalid_client	客户端提供的验证信息有误
unauthorized_client	客户端提供的信息无效
redirect_uri_mismatch	客户端提供的回调地址不一致
access_denied	授权请求被用户或服务器拒绝
access_limited	访问权限受到使用限制，或当前权限达到了最大访问次数
unsupported_response_type	请求的auth授权方式有误
invalid_grant	授权请求有误，或验证码超时、无效
unsupported_grant_type	请求的验证方式有误
invalid_token	请求的access_token无效，同时服务器响应返回401错误
expired_token	请求的access_token超时，同时服务器响应返回401错误
